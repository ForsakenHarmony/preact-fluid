<b>Examples:</b>

1. Icon Sizes:

```js
<div>
    <Icon
        name="bath"
        size="xsmall"
    />
    <Icon
        name="bath"
        size="small"
    />
    <Icon
        name="bath"
        size="normal"
    />
    <Icon
        name="bath"
        size="large"
    />
    <Icon
        name="bath"
        size="xlarge"
    />
</div>
```
